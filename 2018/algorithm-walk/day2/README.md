## 리스트
- List: 자료들이 떨어져 있고, 선으로 연결되어 있는 구조
1. 단방향 리스트(Linked List) : 한 쪽으로만 리스트가 연결된 구조
    - 기본 구조    
        [Data | Link] (단, 노드는 데이터마다 다른 구조를 가진다)
    - 삽입
    - 삭제
2. 양방향 리스트 : 양 쪽으로 리스트가 연결된 구조
    - 기본 구조
        [Link | Data | Link]
    - 삽입
    - 삭제
